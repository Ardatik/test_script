# Генератор отчётов о расходах студентов на кофе

Скрипт считывающий один или несколько csv файлов и выводящий отчёт о медианном потреблении кофе студентами.

## Описание структуры проекта:

main.py - основной файл запуска приложения

src/cli.py: input() - функция для считывания параметров из командной строки
src/cli.py: output_report() -- функция для формирования вывода отчёта в терминал в табличном виде

src/csv_handler.py: read_csv_files() - функция для обработки csv файлов и извлечения данных из поступающих файлов

src/report_handler.py: reports - словарь, сопоставляющий название отчёта с функцией-обработчиком и заголовками

src/median_coffee.py: median_coffee_report() - функция генерации отчёта с медианными расходами

src/stats.py: calculate_median() - функция для вычисления медианы значений

Тесты проверяют корректность парсинга csv файлов, сортировки, вычисления медианы и флагов --files и report.
